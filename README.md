# Playwright Test Automation Framework

Автоматизация тестирования локальной веб-страницы с использованием **Playwright** и **TypeScript**.

## Тестируемое приложение
Все тесты выполняются на локальной странице `index.html`.

### Что есть на странице:
- Форма регистрации
- Валидация полей
- Загрузка файлов
- Модальное окно с результатом

## Требования

- **Node.js** 18 или выше
- **npm**

## Установка

**1. Клонирование репозитория:**
```bash
git clone https://github.com/JoehFlu/playwright-tests.git
cd playwright-tests
```

**2. Установка зависимостей:**
```bash
npm install
```

**3. Установка браузеров Playwright:**
```bash
npx playwright install
```

## Структура проекта

| Файл/Папка | Описание |
|------------|----------|
| `index.html` | Локальная страница с формой |
| `styles.css` | Стили страницы |
| `script.js` | Логика формы |
| `assets/` | Изображения для загрузки |
| `tests/` | Тестовые файлы |
| `tests/basic.spec.ts` | Первый тест |
| `playwright.config.ts` | Конфигурация Playwright |
| `package.json` | Скрипты и зависимости |
| `README.md` | Документация |

## Запуск тестов

**Запуск всех тестов**
```bash
npm test
```

**Запуск в режиме UI**
```bash
npm run test:ui
```

**Запуск конкретного теста**
```bash
npx playwright test tests/basic.spec.ts
```

**Запуск в headed режиме**
```bash
npm run test:headed
```

**Запуск с отладкой**
```bash
npx playwright test --debug
```

## Конфигурация

Конфигурация находится в файле `playwright.config.ts`.

**Основные настройки:**
- Chromium, Firefox и WebKit
- Обычный запуск и headed-режим
- Трассировка при повторной попытке
- HTML-отчёт

## Полезные команды

**Генератор кода для локальной страницы**
```bash
npx playwright codegen file://$(pwd)/index.html
```
